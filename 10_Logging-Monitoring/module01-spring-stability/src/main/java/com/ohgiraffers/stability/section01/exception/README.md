# Section 01: 예외 처리 시스템 구축

온라인 도서관 관리 시스템의 예외 처리를 체계적으로 구현한다.

## 학습 목표
- 커스텀 예외 클래스 설계 (도서 관련 예외들)
- @RestControllerAdvice를 통한 전역 예외 처리
- 일관된 오류 응답 구조 설계
- 비즈니스 예외와 시스템 예외 구분 처리

## 패키지 구조
```
section01/
├── exception/          # 커스텀 예외 클래스들
├── handler/           # 전역 예외 처리 핸들러
├── dto/              # 오류 응답 DTO
├── controller/       # 도서 관리 컨트롤러
├── service/          # 도서 관리 서비스
└── entity/           # 도서 관련 엔티티
```

## 주요 시나리오
- 도서 등록/조회/수정/삭제
- 존재하지 않는 도서 조회 시 예외 처리
- 중복 ISBN 등록 시 예외 처리
- 잘못된 입력값에 대한 예외 처리
